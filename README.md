# Animalium Fabric

A Minecraft Fabric mod that adds aggressive wild mobs to your world.

## Mobs

- **Bear** - Large aggressive mob found in forests
- **Wild Dog** - Pack hunter mob
- **Rat** - Small aggressive mob
- **Piranha** - Water-dwelling aggressive mob

## Features

- Custom entity models and textures
- Special items (pelts, meat, claws)
- Custom armor and tools
- Spawn configuration
- Attack behavior customization

## Building

1. Clone the repository
2. Run `./gradlew build`
3. The mod JAR will be in `build/libs/`

## Installation

Place the built JAR file in your Minecraft `mods` folder.

## Requirements

- Minecraft 1.20.1
- Fabric Loader
- Fabric API
